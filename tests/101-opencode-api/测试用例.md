# 101-opencode-api 测试用例

## 用例列表

| 用例ID | 名称 | 请求 | 期望 |
|---|---|---|---|
| TC-001 | 健康检查 | `GET /health` | 200 + `status` |
| TC-002 | 后端状态扩展 | `GET /x/backends` | 200 + 数组 |
| TC-003 | 路由表扩展 | `GET /x/routes` | 200 + 数组 |
| TC-004 | SSE 状态扩展 | `GET /x/sse` | 200 + connectors |
| TC-005 | 聚合健康 | `GET /global/health` | 后端可用=200，不可用=503/502 |
| TC-006 | 项目列表聚合 | `GET /project` | 后端可用=200，不可用=503/502 |
| TC-007 | 会话列表聚合 | `GET /session` | 后端可用=200，不可用=503/502 |
| TC-008 | 提供商聚合 | `GET /provider` | 后端可用=200，不可用=503/502 |
| TC-009 | 主后端配置 | `GET /config` | 后端可用=200，不可用=503/502 |
| TC-010 | 创建会话 | `POST /session` | 后端可用=200/201，不可用=503 |

## 说明

- `TC-005` 到 `TC-010` 依赖后端 opencode server 状态。
- 自动化脚本会将“后端未启动导致的 503/502”标记为预期降级结果。
